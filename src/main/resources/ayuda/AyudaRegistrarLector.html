Ayuda - Registro de Usuarios

Esta ventana permite registrar nuevos usuarios en el sistema. A continuación, se describen los campos y funcionalidades disponibles:

Campos de Entrada

  • Nombre: Ingrese su nombre.
  • Apellido: Ingrese su apellido.
  • Correo: Debe ingresar un correo válido con formato ejemplo@dominio.com.
  • Contraseña: Ingrese su contraseña.
  • Repetir Contraseña: Ingrese nuevamente la contraseña para confirmarla.

Botones y Funcionalidades

  • Registrar: Guarda los datos ingresados si la información es válida.
  • Cerrar: Cierra la ventana tras una confirmación.
  • Mostrar/Ocultar Contraseña: Al hacer clic en el icono de la contraseña, puede alternar entre mostrar y ocultar la contraseña ingresada.
  • Ayuda: Muestra esta ventana con información sobre el funcionamiento del formulario.

Mensajes de Alerta

  • Registro Exitoso: Se mostrará un mensaje informando que el usuario se ha registrado correctamente.
  • Error en el Registro: Se mostrará un mensaje de error si los datos ingresados son incorrectos o si las contraseñas no coinciden.
  • Confirmación de Cierre: Antes de cerrar la ventana, se preguntará al usuario si realmente desea salir.

Si tiene más dudas, consulte con el administrador del sistema.
