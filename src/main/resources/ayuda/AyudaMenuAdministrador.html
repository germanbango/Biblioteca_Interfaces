Ayuda - Menu Administrador

Este módulo gestiona la interfaz de administración de un sistema de biblioteca. Permite a los administradores realizar diversas operaciones esenciales para la gestión de usuarios y libros.

Las funcionalidades principales incluyen:

  • Gestión de usuarios: Crear nuevos usuarios, modificar sus datos y darlos de baja cuando sea necesario.
  • Gestión de libros: Eliminar libros del sistema según los requerimientos de la biblioteca.
  • Generación de informes: Utiliza JasperReports para generar reportes detallados sobre la biblioteca.
  • Cierre de sesión: Permite a los administradores cerrar sesión y regresar a la pantalla de inicio.
  • Salida del sistema: Opción para cerrar completamente la aplicación.

El módulo está diseñado para garantizar un control eficiente sobre los recursos de la biblioteca, proporcionando una interfaz intuitiva y de fácil acceso para los administradores.
