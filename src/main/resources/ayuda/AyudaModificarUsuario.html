Descripción de la Ventana: Modificar Usuario

La ventana "Modificar Usuario" está diseñada para permitir a los administradores modificar la información de los usuarios existentes en el sistema.

Componentes de la ventana:

  • Tabla de Usuarios: Muestra una lista de todos los usuarios registrados en el sistema. Cada fila de la tabla contiene información básica sobre el usuario, como su correo electrónico y el rol asignado.
  • Nombre: Permite al administrador editar el nombre del usuario seleccionado.
  • Apellido: Este campo permite modificar el apellido del usuario seleccionado.
  • Correo Electrónico: Aquí se puede cambiar el correo electrónico del usuario. El sistema buscará el usuario por el correo electrónico para realizar la modificación.
  • Contraseña: Permite modificar la contraseña del usuario seleccionado.
  • Rol: Aquí se puede cambiar el rol del usuario, eligiendo entre los diferentes tipos de usuario disponibles, como "Administrador" o "Usuario Normal".

Botones de la ventana:

  • Modificar Usuario: Este botón se presiona para aplicar los cambios realizados al usuario seleccionado. Después de modificar los datos, el sistema guarda los cambios y muestra una notificación de éxito.
  • Cerrar: Este botón cierra la ventana. Antes de cerrarse, se muestra una confirmación para asegurarse de que el administrador realmente desea cerrar la ventana.

Mensajes de la ventana:

  • Modificación Completa: Si el usuario fue modificado correctamente, se muestra un mensaje que dice "Usuario modificado correctamente". Este mensaje confirma que los cambios fueron guardados.
  • Confirmación de Cierre: Al intentar cerrar la ventana, el sistema pregunta al administrador si está seguro de que desea cerrarla. Si se confirma, la ventana se cierra.

Flujo de trabajo:

  1. El administrador selecciona un usuario de la tabla de usuarios.
  2. Se cargan los datos de ese usuario en los campos de texto correspondientes.
  3. El administrador modifica los campos que desee cambiar (nombre, apellido, correo electrónico, contraseña, rol).
  4. Después de realizar los cambios, el administrador hace clic en el botón "Modificar Usuario".
  5. Si la modificación es exitosa, se muestra un mensaje de confirmación y los datos en la tabla se actualizan.
  6. Si el administrador desea cerrar la ventana, puede hacer clic en el botón "Cerrar", y se le pedirá confirmación antes de que la ventana se cierre.

Esta ventana permite a los administradores modificar la información de los usuarios de manera eficiente y fácil, asegurando que el sistema mantenga la información actualizada y precisa.
