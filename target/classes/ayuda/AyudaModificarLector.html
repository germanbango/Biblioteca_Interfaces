Guía de Uso - Modificar Cliente

Esta ventana permite a los clientes modificar sus datos personales dentro del sistema.

Funciones Disponibles

  • Modificar Datos: Los campos permiten actualizar el nombre, apellidos, correo y contraseña.
  • Confirmación: Una vez actualizados los datos, se muestra un mensaje de éxito.
  • Cerrar Ventana: Opción para cerrar la ventana con una confirmación previa.
  • Ayuda: Al hacer clic en el icono de ayuda, se muestra esta guía.

Pasos para Modificar la Información

  1. Ingresar los nuevos datos en los campos correspondientes.
  2. Presionar el botón de Modificar para guardar los cambios.
  3. Si la modificación es exitosa, aparecerá una notificación de confirmación.
  4. Para cerrar la ventana, presionar el botón de Cerrar, lo que pedirá confirmación antes de salir.

Errores Comunes

  • Si algún campo está vacío, la modificación no se realizará.
  • El correo debe tener un formato válido (ejemplo@correo.com).

Si tiene dudas adicionales, comuníquese con el soporte técnico.
