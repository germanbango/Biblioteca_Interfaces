Guía de Uso - Menú Cliente

Esta ventana corresponde al menú principal del cliente dentro del sistema. Desde aquí, el usuario puede realizar diversas acciones relacionadas con la gestión de su cuenta y los préstamos de libros.

Opciones del Menú

  • Modificar Usuario: Permite al usuario editar su información personal.
  • Solicitar Préstamo: Opción para solicitar el préstamo de un libro.
  • Devolver Libro: Facilita la devolución de un libro prestado.
  • Cerrar Sesión: Cierra la sesión actual y redirige a la pantalla de inicio de sesión.
  • Salir: Cierra la aplicación completamente.

Instrucciones de Uso

Para utilizar el menú, simplemente haga clic en la opción deseada y siga las instrucciones en pantalla.

Detalles Adicionales

El nombre del usuario aparecerá en la parte superior del menú. La aplicación ha sido diseñada para facilitar la gestión de préstamos de libros de manera sencilla y eficiente.
