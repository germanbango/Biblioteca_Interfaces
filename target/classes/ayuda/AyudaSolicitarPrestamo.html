Guía de Uso - Ventana de Solicitud de Préstamo

Esta ventana permite a los usuarios solicitar el préstamo de un libro de la biblioteca.

Funciones Principales

  • Selección de libro: Permite elegir un libro de la lista desplegable.
  • Visualización de información: Muestra detalles como el autor, género, año de publicación, disponibilidad y fecha de devolución.
  • Generar préstamo: Confirma el préstamo del libro seleccionado si está disponible.
  • Cerrar ventana: Cierra la ventana actual.

Pasos para Solicitar un Préstamo

  1. Selecciona un libro del menú desplegable.
  2. Revisa la información del libro, incluyendo su disponibilidad.
  3. Si el libro está disponible, presiona el botón de "Generar Préstamo".
  4. Si el préstamo se realiza con éxito, aparecerá un mensaje de confirmación.
  5. Si deseas salir, presiona el botón de cerrar.

Errores Comunes

  • No se muestra información del libro: Asegúrate de haber seleccionado un libro del listado.
  • Libro no disponible: Intenta con otro título o espera a que el libro se devuelva.
  • No se genera el préstamo: Contacta con el administrador si el problema persiste.

Nota: La fecha de devolución del libro se establece automáticamente en 15 días después del préstamo.
