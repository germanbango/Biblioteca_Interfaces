Descripción de la Ventana: Crear Usuario

La ventana "Crear Usuario" está diseñada para permitir a los administradores agregar nuevos usuarios al sistema.

Campos de la ventana:

  • Nombre: Aquí se ingresa el primer nombre del usuario.
  • Apellido: Este campo es para el apellido del usuario.
  • Correo Electrónico: Es necesario proporcionar un correo electrónico válido para el usuario. El sistema validará que el formato del correo sea correcto (por ejemplo, ejemplo@dominio.com).
  • Contraseña: Se debe ingresar una contraseña para el usuario. Es importante que la contraseña sea segura y que se recuerde para el futuro.
  • Rol: En este campo se selecciona el tipo de usuario que se está creando, como "Administrador" o "Usuario Normal". Esto define los permisos que tendrá el nuevo usuario en el sistema.

Botones de la ventana:

  • Crear Usuario: Este botón se presiona para guardar el nuevo usuario en el sistema. Si los datos son válidos (como un correo electrónico en el formato correcto), el usuario será creado. Si hay algún error, como un correo incorrecto, aparecerá un mensaje de error.
  • Cerrar: Este botón cierra la ventana actual. Antes de cerrarse, se muestra una confirmación para asegurarse de que el usuario realmente desea cerrar la ventana.

Mensajes de la ventana:

  • Registro Completo: Si el usuario fue creado correctamente, aparecerá un mensaje de éxito que dice "Usuario Registrado correctamente".
  • Error: Si el registro falla, aparecerá un mensaje de error que indica "Fallo en el registro", con detalles de que los datos ingresados no son correctos.

Esta ventana permite al administrador agregar nuevos usuarios de manera eficiente y segura al sistema, con alertas claras para informar sobre el éxito o cualquier error en el proceso.
